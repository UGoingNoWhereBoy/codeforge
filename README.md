# Get Started
### 1. Clone the repository:
```bash
git clone https://github.com/UGoingNoWhereBoy/codeforge
```
### 2. CD into the repository
```bash
  cd codeforge
```
### 3. Install packages 
```bash
npm install
```
### 4. Create a .env file and fill it with your own api key from openai
```bash
touch .env.local
```
### 5. Run the app 
```bash
npm run dev
```
### 6. Give it a star
